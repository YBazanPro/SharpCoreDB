# SharpCoreDB.Graph.Advanced v1.5.0

Advanced graph analytics and GraphRAG for SharpCoreDB on .NET 10.

## Key Features

- Community detection with Louvain, Label Propagation, and Connected Components
- Centrality metrics including Degree, Betweenness, Closeness, Eigenvector, and Clustering Coefficient
- Subgraph analysis with K-core, clique, and triangle queries
- SQL-callable graph analytics helpers
- GraphRAG ranking that combines semantic and topological signals
- Result caching and performance profiling utilities

## Installation

```bash
dotnet add package SharpCoreDB.Graph.Advanced --version 1.5.0
```

## Package Dependencies

`SharpCoreDB.Graph.Advanced` works with:

- `SharpCoreDB`
- `SharpCoreDB.Graph`
- `SharpCoreDB.VectorSearch`
